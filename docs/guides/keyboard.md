# Keyboard reference

| Shortcut | Action | Context |
| --- | --- | --- |
| `⌘P` | Open command palette | Workspace |
| `⌘↵` | Run editor SQL | Query editor |
| `Esc` | Close the current overlay or return | Modal or workspace |

The command palette includes tables and common workspace actions. Open it, type part of a table name or action, then choose the result.

## Efficient workflow

1. Use `⌘P` to jump to a table.
2. Apply filters before browsing a large dataset.
3. Stage related cell changes.
4. Review the generated SQL once.
5. Commit the batch.
6. Move to the Query tab and use `⌘↵` for ad hoc checks.

Native controls also follow standard macOS text navigation, selection, copy, paste, and focus behavior where supported by gpui and guise.
